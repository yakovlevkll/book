# Степени и корни

# Основные

Определение степени и корня

bx=a<=>ax=bb^x = a <=> \sqrt[x]{a} = bbx=a<=>xa​=b

Умножение и деление

bx⋅by=bx+yb^x * b^y = b^{x + y}bx⋅by=bx+y
bx:by=bx−yb^x : b^y = b^{x - y}bx:by=bx−y

Возведение степени в степень

(bx)y=bx⋅y(b^x)^y = b^{x * y}(bx)y=bx⋅y

Корень

bxy=bxy\sqrt[y]{b^x} = b^{\frac{x}{y}}ybx​=byx​

Отрицательная степень

a−x=1bxa^{-x} = \dfrac{1}{b^x}a−x=bx1​

Одинаковый показатель степени

ax⋅bx=(a⋅b)xa^x * b^x = (a * b)^xax⋅bx=(a⋅b)x
axbx=(ab)x\dfrac{a^x}{b^x} = (\dfrac{a}{b})^xbxax​=(ba​)x
