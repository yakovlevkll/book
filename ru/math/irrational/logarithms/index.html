# Логарифмы

Выражение

bx=ab^x = abx=a

может быть переписано не только в привычной форме с использованием корня

ax=b\sqrt[x]{a} = bxa​=b

Но и с использованием логарифма

log⁡ba=x\log_b a = xlogb​a=x

# Разновидности логарифмов

Натуральный логарифм -- логарифм с основанием, равным константе eee

ln⁡x=log⁡ex\ln x = \log_e xlnx=loge​x

Десятичный логарифм -- логарифм с основанием, равным 10

lg⁡x=log⁡x=log⁡10x\lg x = \log x = \log_{10} xlgx=logx=log10​x

# Основные правила

Правила ниже сформулированы для основания bbb, которое должно удовлетворять следующим условиям

b≠1, b>0b \ne 1,\ b > 0b=1, b>0

Основное свойство логарифма

blog⁡ba=ab^{\log_b a} = ablogb​a=a

Правило кажется странным, но стоит только вдуматься, как оно становится очень логичным -- при возведении числа bbb в такую степень, которая при возведении этого же числа дает число aaa, в результате даёт это самое aaa.

Логарифм произведения

log⁡b(x⋅y)=log⁡bx+log⁡by\log_b (x * y) = \log_b x + \log_b ylogb​(x⋅y)=logb​x+logb​y

Логарифм частного

log⁡b(xy)=log⁡bx−log⁡by\log_b ( \dfrac{x}{y} ) = \log_b x - \log_b ylogb​(yx​)=logb​x−logb​y

Переход к новому основанию. Новое основание должно удовлетворять тем же правилам, что и старое -- быть больше нуля и не равняться единице

log⁡bx=log⁡cxlog⁡cb\log_b x = \dfrac{\log_c x}{\log_c b}logb​x=logc​blogc​x​

  2. log⁡bkx=1klog⁡bx\log_{b^k} x = \dfrac{1}{k} \log_b xlogbk​x=k1​logb​x
